Privacy Policy

JSON Workbench Privacy Policy

Last updated: 2026-07-04

JSON Workbench is a local-first Chrome extension for repairing JSON and generating developer artifacts.

Summary

JSON Workbench does not send your data to any server. It does not use AI services, telemetry, analytics, advertising SDKs, or remote APIs.

Data Processed

The extension may process text you type, text you explicitly paste, selected page text sent through the context menu, and files you explicitly import.

Local History

JSON Workbench can save your original input and repaired JSON locally in your browser. History is stored only on your device. Private mode pauses history saving.

Network Access

JSON Workbench is designed not to make network requests. It does not transmit your JSON, history, generated outputs, clipboard contents, or selected page text.

Third Parties

JSON Workbench does not sell, share, or transfer user data to third parties.

Contact

Use the GitHub issue tracker for support or privacy questions.
