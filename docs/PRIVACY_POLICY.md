# JSON Workbench Privacy Policy

Last updated: 2026-07-04

JSON Workbench is a local-first Chrome extension for repairing JSON and generating developer artifacts.

## Summary

JSON Workbench does not send your data to any server. It does not use AI services, telemetry, analytics, advertising SDKs, or remote APIs.

All JSON repair, formatting, schema generation, code generation, mock data generation, and history storage happen locally in your browser.

## Data Processed

JSON Workbench may process:

- Text you type into the extension.
- Text you explicitly paste from your clipboard by clicking the Paste button.
- Text you explicitly send from a web page by using the "Fix selected JSON" context menu item.
- Files you explicitly import into the extension.

## Local History

JSON Workbench can save your original input and repaired JSON locally in your browser so you can revisit recent work.

History is stored only on your device. It is not uploaded or shared.

You can:

- Enable Private mode to pause history saving for the current session.
- Delete individual history items.
- Clear all local history.
- Clear entries marked as sensitive.

## Network Access

JSON Workbench is designed not to make network requests. It does not transmit your JSON, history, generated outputs, clipboard contents, or selected page text.

## Third Parties

JSON Workbench does not sell, share, or transfer user data to third parties.
