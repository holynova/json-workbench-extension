# Store Asset Inventory

- `icons/store-icon-128.png`, 128x128, Chrome Web Store icon.
- `../public/icons/icon-16.png`, 16x16, extension toolbar icon.
- `../public/icons/icon-32.png`, 32x32, extension icon.
- `../public/icons/icon-48.png`, 48x48, extension icon.

## Source Render Pages

The HTML sources used to render icons and promotional images are in `render/`.

Icons are generated by `scripts/generate-icons.mjs` to guarantee exact PNG dimensions.
